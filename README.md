# DrawPoker

## 概要
Unityで制作したオンライン対戦型のポーカーゲームです。

Photon Unity Networkingを用いて複数人で同じルームに入り、カード配布、ターン進行、ベット、勝敗判定までを行えるように実装しました。

本作品は、研究で作成した脳波共有システムの前段階として制作した、脳波共有機能を含まない公開用のゲーム作品です。

## 特徴
- Unityによるポーカーゲーム実装
- Photonを用いたマルチプレイ通信
- プレイヤーごとの手札、場札、チップ、ポットの管理
- チェック、コール、レイズ、フォールド、オールインなどの操作
- ポーカー役の判定処理
- UI上でのゲーム状態表示

## 主な実装
- MultiplayerGameManager.cs
  - ゲーム進行、ターン管理、ベット管理、カード配布、プレイヤー状態の同期を担当
- PokerHandEvaluator.cs
  - 役判定、キッカー比較、7枚から最良の5枚を選ぶ処理を担当
- PokerUIManager.cs
  - ボタン操作やチップ、ポット、カード表示などのUI制御を担当
- PhotonLauncher.cs
  - Photonへの接続、ルーム参加、ゲーム開始処理を担当

## 工夫した点
ポーカーは、カードの強さだけでなく、ターン進行、ベット額、フォールド状況、ポット管理など複数の状態が同時に変化します。

そのため、ゲーム進行を管理する処理と、役判定・UI表示・通信処理を分け、複数人で遊んだときにも状態が破綻しないように意識して実装しました。

## 研究版との関係
本作品をもとに、別プロジェクトとして脳波共有機能を組み込んだ研究版を開発しました。

研究版では、プレイヤーの脳波情報を受信し、ゲーム中に可視化・共有する仕組みを追加しています。ただし、脳波データ、被験者情報、共有機能の詳細実装については、研究倫理および所属研究室の方針を考慮し、公開していません。
